# Front-end (Casare Motos)

Este diretório contém a aplicação Front-end feita com Vite + React + TypeScript.

Como rodar localmente:

```powershell
cd 'C:\Users\renan\Projetos\Loja_Motos\Front-end'
npm install
npm run dev
```

Páginas disponíveis:
- `/` — Home
- `/compra` — Compra (lista de anúncios)
- `/venda` — Venda (instruções / formulário futuro)

Design:
- Nome do site: **Casare Motos**
- Paleta de cores definida em `src/index.css`.
- Logo em `src/logo.svg` (simples, vetorial).

Observações:
- A pasta `Back-end` existe como placeholder; por enquanto não há servidor.
